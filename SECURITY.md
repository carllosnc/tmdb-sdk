# Security Policy

## Supported Versions

Currently, only the latest release of the TMDB SDK is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.x.x   | :white_check_mark: |

## Reporting a Vulnerability

We take the security of this project seriously. If you discover a security vulnerability in this package, please **DO NOT** report it by creating a public GitHub issue. 

Instead, please report vulnerabilities by opening a **Private Vulnerability Report** on GitHub.

1. Go to the [Security advisory page](https://github.com/carllosnc/tmdb-sdk/security/advisories) of this repository.
2. Click on **Report a vulnerability**.
3. Provide as much information as possible, including steps to reproduce the issue.

We will acknowledge receipt of your vulnerability report and strive to send you regular updates about our progress. If a fix is needed, we will release a patch as quickly as possible.
